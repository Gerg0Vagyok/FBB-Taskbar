## Compiling
### Dependencies
- Make
- Bear
- Clang
- QT 6
- QT 6 for wayland
- Layer Shell QT

### Arch Linux

	# pacman -S make bear clang qt6-base layer-shell-qt qt6-wayland
	$ make compile
